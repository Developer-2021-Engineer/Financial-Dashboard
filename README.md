# Financial-Dashboard
A modern financial dashboard made with React and Javascript.

# Step 1:
You have to download the entire zip file form this repository.

# Step 2:
i) You will see 5 parts of zip files aling with a Readme file. Incase you don't see the Readme file don't worry it is only to read and not part of the dashboard. 
ii) Next use an online file joining tool like "https://www.toolsley.com/split.html" to join the 5 part zip file.

# Step 3:
i) Once combined download that combined file.
ii) Now extract that combined file. You will see a Modern Financial Dashboard file.
iii) Open that file in a code editor like VScode, etc. that you use. 
# Note: VScode is recommended.

# Step 4:
i) Install all these dependencies:
# Initialize a new React project using Vite
npm create vite@latest . -- --template react

# Install the core React and Vite dependencies
npm install

# Install UI icons (Lucide), Charting (Recharts), and Date utilities (date-fns)
npm install lucide-react recharts date-fns

# Start the local development server to view the app
npm run dev

# Build the project into an optimized 'dist' folder for production
npm run build

# Step 5:
With this your program should be functioning. If not go to the localhost page and Inspect it. If there is an error copy past the error in an AI agent like Gemini and ask its assistance.
